Action Required: Configuration Needed

This RSVP page is not connected to a database yet. To make it work, you need to paste your unique configuration keys from Firebase.

  1. Go to the Firebase Console and create a new project (it's free).
  2. In your new project, add a new Web App (click the </> icon).
  3. Firebase will give you a firebaseConfig object. Copy this entire object.
  4. Open this index.html file in a text editor and paste the object you copied into the designated area in the script section below.
